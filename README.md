# 2048
2048 GAME
2048 Game
This project is an implementation of the popular 2048 puzzle game.
The game is played on a grid where the player can move tiles in four directions.
When two tiles with the same number collide, they merge into one with double the value.
The objective is to reach the 2048 tile while managing limited space on the board.

Features:

Grid-based movement

Tile merging logic

Score calculation

Game over detection
